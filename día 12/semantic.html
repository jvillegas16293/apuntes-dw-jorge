Elementos con significado

Ejemplos de etoquetas semanticas: Table, form, article. Ejemplos etiquetas NO semanticas: div, span (img web semantica)

que es un header, cuando utilizamos esta etiqueta

Es el encabezado de una págna, sección o artículo. incluye elementos com el logo, texto introductorio y incluso menús de navegación. Puede haber multiples header en una página, pero no puede haber un header en un footer u otro header.

Footer

Es el pie de página de un sitio o de una seccion, puede haber multiples por página y puede incluir información como (que espera encontrar google en el Footer) - Información de contacto. - Enlaces a las RRSS. - Creditos - Forulario de contacto

La navegación (nav)

Define un conjunto de enlaces de navgación. No todos los enlaces deben estar dentro de un "nav". Puede estar donde se necesite.

Main

Crea un contenedos para el contenido pricipal de nuestra página. No puede haber dos main en la misma página y no debe incluir contenido repetido de otras secciones, del sidebar, logos, ect... No puede ser hijo de etiquetas como: article, footer, aside, header o nav.

Aside

Define el contenido relacionado inderectamente con el contenido principal, como por ejemplo un sidebar. Elementos que solemos poner en un "aside"
  • Publicidad
  • Biografía del autor
  • Enlaces relacionados
  • Información de perfil

Section

Define una seccion en una página, donde se agrupan los contenidos relacionados y normalemente incluyen un encabezado. Una web, se compone de varias secciones cada una con su contenido y propósito
  • Capitulos
  • Introduucín
  • Categorias de noticias
  • Información de contacto

Article

Define un contenido independiente, el mismo puede ser compartido a reutilizado en otro sitio sin requerir contexto, tambien suele incluir un título
  • Post de un blog
  • Un artículo de una revista
  • Comentario de un usuario (reseñas)
  • Tarjeta de producto

Puede encontrar artículos dentro de secciones y secciones dentro de artíulos. Un artículo puede tener multiples secciones una sección puede tener multiples artículos.

Fgure

Define elmentos autocontenidos, como pueden ser una imagen, un diagráma, un gráfico una tabla o una foto, ect. Dentro de ella, se sule utilzar la etiqueta "fifcaption" para describir el contenido, es es además ua incluido "alt" en caso de imágenes. pone una descripcion visible para el usuario.

Mark
